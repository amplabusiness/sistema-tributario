## 🔧 **INSTALAÇÃO E DESENVOLVIMENTO**

### **Pré-requisitos:**
- Node.js 18.x+
- npm ou yarn
- Git

### **Setup Local:**
```bash
# Clone o repositório
git clone https://github.com/seu-usuario/sistema-tributario.git
cd sistema-tributario

# Backend
cd backend
npm install
npm run dev

# Frontend (novo terminal)
cd frontend
npm install
npm run dev
```

### **Variáveis de Ambiente:**
```bash
# Backend (.env)
DATABASE_URL="postgresql://..."
REDIS_URL="redis://..."
OPENAI_API_KEY="sk-..."
JWT_SECRET="..."

# Frontend (.env.local)
NEXT_PUBLIC_API_URL="http://localhost:3001"
```
